# Cohort 3.0 🎓

A comprehensive web development learning project showcasing assignments and tasks completed during Cohort 3.0. This repository demonstrates practical implementation of modern web technologies and development best practices.

## 📋 Overview

This repository contains a curated collection of web development exercises, projects, and assignments with a focus on:
- **Frontend Technologies**: HTML5, CSS3, JavaScript (ES6+)
- **Responsive Design**: Mobile-first approach and cross-browser compatibility
- **Best Practices**: Clean code, accessibility standards, and semantic markup
- **Version Control**: Professional Git workflow and collaboration techniques

## 🚀 Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, or Edge)
- Git installed on your system
- Text editor (VS Code recommended)

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/nishantb01/cohort-3.0.git
   cd cohort-3.0
   ```

2. **View projects locally:**
   - Open any HTML file directly in your browser
   - Or use Live Server extension in VS Code for better development experience

3. **Explore assignments:**
   - Navigate to the `A2`, `A3`, or `task1` folders
   - Each contains independent projects with their own structure

## 🐛 Troubleshooting

### Projects not displaying?
- Ensure you're using a modern web browser
- Check browser console for errors (F12)
- Verify file paths in HTML links

### Changes not reflecting on GitHub Pages?
- Wait 1-2 minutes for GitHub to rebuild the site
- Clear browser cache (Ctrl+Shift+Delete or Cmd+Shift+Delete)
- Verify your default branch is set to `main` or `master`
